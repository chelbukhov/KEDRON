"# KEDRON" 
Смарт-контракт токена и телеграм бот разработаны для учета вкладов участников поселений в различные проекты.
Токен служит только для учета и не является платежным. Также бот позволяет участникам поселений видеть различные проекты в других поселениях,
контактировать с их администраторами, а также участвовать в этих проектах в качестве инвесторов.
